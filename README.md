# 뉴스피드 프로젝트
> 이 프로젝트는 회원가입 및 로그인 후 포스트와 댓글을 작성하고 조회할 수 있는 뉴스피드 애플리케이션입니다. 사용자는 다른 사용자를 팔로우할 수 있으며, 포스트 및 댓글에 좋아요를 남길 수 있습니다. 또한, 자신의 포스트 및 댓글을 수정하거나 삭제할 수 있고, 프로필 조회 기능도 제공합니다.

<br>

## 주요 기능

### 1. 회원 관리

- 회원가입

- 로그인 / 로그아웃

- 프로필 조회

### 2. 포스트 (게시글) 관리

- 포스트 작성

- 포스트 수정

- 포스트 삭제

- 전체 포스트 조회 (페이지네이션 적용)

- 팔로우한 유저의 포스트 조회

### 3. 댓글 관리

- 댓글 작성

- 댓글 수정

- 댓글 삭제

- 특정 포스트의 댓글 조회

### 4. 좋아요 기능

- 포스트 좋아요 및 취소

- 댓글 좋아요 및 취소

### 5. 팔로우 기능

- 유저 팔로우 및 팔로우 취소

- 팔로우한 유저의 포스트 목록 조회
